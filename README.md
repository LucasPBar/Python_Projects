# 📂 Python Projects

- **📚 Project Biblioteca** – Sistema simples para gerenciar livros em uma biblioteca.

## 🛠 Tecnologias Utilizadas

- **Python** 🐍

- **Jupyter Notebook** para organização dos projetos
